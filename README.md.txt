# 📅 Interactive Calendar Component

## 🚀 Overview

This project is a **responsive and interactive calendar component** built using **Next.js (React)**.
It is inspired by a physical wall calendar and focuses on combining **clean UI design with practical functionality**.

The goal was to translate a static design into a **fully functional frontend component**, while maintaining simplicity, usability, and responsiveness.

---

## ✨ Features

* 📆 **Wall Calendar UI**

  * Hero image section with a clean monthly layout
  * Structured grid with proper weekday alignment

* 🎯 **Date Range Selection**

  * Select a **start date** and **end date**
  * Visual highlighting for:

    * Start & End dates
    * Dates in between

* 📝 **Notes Section**

  * Add notes for planning or reminders
  * Notes are saved using **localStorage**

* 🔄 **Month Navigation**

  * Easily switch between months

* 🎨 **Modern UI**

  * Styled using Tailwind CSS (utility-based styling)
  * Clean, minimal, and user-friendly interface

* 📱 **Fully Responsive**

  * Works smoothly on desktop and mobile devices

---

## 🌟 Additional Enhancements

To go beyond the basic requirements, the following improvements were added:

* 🎬 Smooth transition animation while switching months
* 📍 Highlight **today’s date**

## 🛠 Tech Stack

## ▶️ How to Run Locally

1. Clone the repository:

```bash
git clone https://github.com/your-username/calendar-app.git
```

2. Navigate to the project:

```bash
cd calendar-app
```

3. Install dependencies:

```bash
npm install
```

4. Run the development server:

```bash
npm run dev
```

5. Open in browser:

```
http://localhost:3000
```
